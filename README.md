# Weather App 🌤️

Простое и элегантное приложение для просмотра погоды в любом городе мира. Создано с использованием HTML, CSS и JavaScript.

## 📱 Скриншоты
![Weather App скриншот](/screenshot.jpg)

## ✨ Функционал

- 🔍 Поиск погоды по названию города
- 🌡️ Отображение текущей температуры
- 💨 Информация о скорости ветра
- 💧 Данные о влажности воздуха
- ☁️ Визуальное отображение состояния погоды (иконки)
- 💾 Сохранение последнего просмотренного города в localStorage
- 📅 Актуальная дата в интерфейсе
- 🎨 Современный дизайн с hover-эффектами
- ✨ Плавные анимации при появлении новых данных
- 🎭 Эффекты наведения на элементах интерфейса

## 🛠️ Технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации и transition)
- JavaScript (ES6+)
- OpenWeatherMap API
- LocalStorage для хранения данных
